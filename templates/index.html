Welcome to OptimiseIt

This tool is designed to assist researchers and students in performing standard statistical analysis and curve fitting for academic papers.

Data Entry Guidelines

  • Format: All data arrays must be separated by commas. (e.g., 12.5, 14.1, 16.0)
  • Decimals: Use periods (.) for decimals, not commas.
  • Equal Lengths: For Regression, T-Test, and Correlation, ensure both input sets have the exact same number of data points.
